Coding

There are several skills that i believe you need to have if you want to be a good programmer :

  • The ability to learn concepts and apply them to the other problems
  • Basic mathematical skills
  • A passion of problem solving
  • Good communication skills

Responsive Website

Without further ado , lets get to how to build an responsive web page :

  • Add responsive meta tags in your HTML document
  • Apply media queries to your layout
  • Make images and embedded videos responsive
  • Ensure your typography will be easily readable on mobile devices

Cross-Browser Development

In some how you should go through these steps until mastering it :

  • Using mobile/desktop browser emulator for each browser
  • Use css reset stylesheet
  • Use frameworks like bootstrap to guarantee cross-compatibility
  • Using a cloud-base platform to perform cross browser testing on browsers installed on real devices like lambda test

Web Design

You don't always need qualifications to become a web designer , but you'll usually need to show that you have got skills in :

  • Visual design
  • UX (user experience)
  • SEO (search engine optimization)
  • Using coding software like HTML and CSS
